# HYP-001 — Capability Table v1 is sufficient for initial agent design

## Claim
Level 1 capability table (fields: CapabilityId, Description, Scope, Inputs, Outputs, Preconditions, Postconditions, Evidence, F-G-R, DecayWindow, Status) is sufficient to start applied-agent design.

## Context (A.1.1)
Meta-agent context only; no applied-domain content here.

## MethodDescription (A.3.2)
- Input: applied agent brief + Level 1 capability table format
- Procedure: derive capabilities for one applied agent; attempt to map sources to evidence refs
- Output: filled capability table + evidence stubs

## Evidence plan (A.10 / B.3 / B.3.4)
- Evidence items: user feedback on adequacy + source mapping viability
- F-G-R: initially low, updated after experiment

## Status
proposed
